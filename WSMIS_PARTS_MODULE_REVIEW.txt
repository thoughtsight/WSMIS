================================================================================
PART 2: BUSINESS PURPOSE
================================================================================

The Parts module covers three reports forming one business domain:

PARTS EXECUTIVE
  - Executive summary of parts revenue performance
  - Mirrors the Labour Executive page in structure
  - High-level KPIs: Revenue, Growth, Load (Job Cards), Margin %
  - Category breakdown: Standard Parts, Oil, Add-ons (Accessories+Tyres+Battery+Other)
  - Trend chart with cross-filter (click bar to filter by month)
  - Location growth ranking chart
  - Executive Location Performance table with rank movement
  - Monthly Location Detail (expandable)

PARTS DETAIL
  - Operational drill-down into parts revenue
  - Category revenue by location (6 categories: Standard Parts, Oil, Accessories, Tyres, Battery, Other)
  - Category mix stacked bar chart (CP vs PP)
  - Service type contribution table (Free/Paid/Running Repair/Accidental)
  - Discount analysis by location

SALES MIX
  - Product mix analysis across all non-labour categories
  - Oil, Tyres, Batteries, Accessories, Standard Parts
  - KPIs for revenue and quantity per category
  - Monthly trend charts per category
  - Location ranking charts
  - Oil revenue per litre analysis

  def _get_metric(df, col_name, aggregate=True):
      """Helper to safely extract and optionally sum a column."""
      if col_name not in df.columns:
          return 0.0 if aggregate else pd.Series(dtype=float, ...)
      s = df[col_name]
      return float(s.sum()) if aggregate else s

  def safe_divide(numerator, denominator, fill_value=0.0):
      """Safely divide, returning fill_value on division by zero."""
      if isinstance(numerator, (pd.Series, pd.DataFrame)) or isinstance(denominator, (pd.Series, pd.DataFrame)):
          res = numerator / denominator
          return res.replace([np.inf, -np.inf], np.nan).fillna(fill_value)
      if denominator == 0 or pd.isna(denominator):
          return fill_value
      return numerator / denominator

  def calc_ratio(numerator, denominator, multiplier=1.0, fill_value=0.0):
      """Calculates ratio (e.g. percentage if multiplier=100)"""
      return safe_divide(numerator, denominator, fill_value) * multiplier

  def calc_growth_pct(current, prior, fill_value=0.0):
      """Calculates YoY or MoM growth percentage."""
      return safe_divide(current - prior, prior, fill_value) * 100
