# Feature Request Template

Please fill out this template to suggest a new feature or dashboard enhancement for WSMIS.

## 1. Description of the Request
Provide a clear and concise description of the new feature. What problem does it solve?

## 2. Target Audience / Roles
Who will use this feature? (e.g., Service Advisors, Dealership Managers, System Admins)

## 3. Mathematical Definitions / Rules (If Applicable)
If requesting a new KPI or analytical block, clearly define the mathematical rules:
- Formula (e.g., `Net Labour = Gross - Discount`)
- Dataset Required (e.g., Service Sheet, Audit Sheet)

## 4. Proposed Layout / Location
Where should this feature be placed? (e.g., A new section in the `Overview` dashboard, or an entirely new Page).

## 5. Alternatives Considered
A clear and concise description of any alternative solutions or features you've considered.

## 6. Priority / Business Value
- [ ] Critical (Unblocks core business operations)
- [ ] High (Significant efficiency gain)
- [ ] Medium (Nice to have, marginal gain)
- [ ] Low (Cosmetic/Edge case)

## 7. Additional Context
Add any other context, wireframes, mockups, or screenshots about the feature request here.
